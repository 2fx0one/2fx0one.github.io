# install
    默认分支是hexo
    
    ```sh
    git clone https://github.com/2fx0one/2fx0one.github.io
    cd 2fx0one.github.io
    npm install hexo
    hexo init
    npm install
    npm install hexo-deployer-git

    #next themes
    git clone https://github.com/theme-next/hexo-theme-next themes/next
    
    ```
